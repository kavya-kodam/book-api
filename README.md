# Book API

A simple REST API for managing books using Node.js and Express.js.

## Features
- View all books (GET /books)
- Add a book (POST /books)
- Update a book (PUT /books/:id)
- Delete a book (DELETE /books/:id)
- Get book by ID (GET /books/:id)
